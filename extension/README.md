# JLCEDA MCP Bridge Extension

This folder contains the source files for the EasyEDA Pro / JLCEDA Pro extension bridge.

For a production `.eext` package, create an EasyEDA Pro extension project with the official
`pro-api-sdk`, copy these files into it, run `npm run build`, then import the generated
`.eext` file in the editor.

Official docs:

- Extension API guide: https://prodocs.easyeda.com/en/api/guide/
- Extension configuration: https://prodocs.easyeda.com/en/api/guide/extension-json.html
- WebSocket API: https://prodocs.easyeda.com/en/api/reference/pro-api.sys_websocket.register.html
